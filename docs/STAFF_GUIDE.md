# מדריך לצוות המטפלים

ברוכים הבאים! המערכת עוזרת לכולנו לדעת מי באיזה חדר, למצוא חדר פנוי,
לדווח היעדרויות ולסדר החלפות — בלי קבוצות וואטסאפ אינסופיות 🙂

---

## כניסה ראשונה

1. קיבלתם מהניהול **שם משתמש** ו**סיסמה זמנית**.
2. היכנסו לכתובת המערכת (קיבלתם קישור) והתחברו.
3. תתבקשו לקבוע **סיסמה אישית** — שמרו אותה! אם תישכח, הניהול יאפס.

### מומלץ: להוסיף למסך הבית של הטלפון

- **iPhone:** פתחו את הכתובת ב-Safari → כפתור שיתוף → "הוסף למסך הבית".
  (חובה בשביל התראות באייפון!)
- **Android:** פתחו ב-Chrome → תפריט ⋮ → "הוספה למסך הבית".

מעכשיו זה מרגיש כמו אפליקציה רגילה.

## המסכים

### הלו"ז שלי
מה מחכה לכם השבוע: החדרים והשעות שלכם, יום אחרי יום. כולל שיבוצים קבועים,
הזמנות חד-פעמיות, ומה שפונה בגלל היעדרות.

### לוח מלא
כל החדרים וכל המטפלים, לכל יום. **לחיצה על משבצת פנויה** פותחת שריון מהיר —
בוחרים שעות ולוחצים. אם החלון נתפס שנייה לפניכם — תקבלו הודעה ותוכלו לבחור
שעה אחרת. יש גם חיפוש: "איפה יוסי עכשיו?".

### הזמנת חדר
אשף חכם למציאת חדר:

1. בוחרים סוג טיפול (רגיל / זוגי-משפחתי / קבוצה), תאריך ושעות,
   והעדפות (חלון, כיור, חדר גדול).
2. המערכת מציעה עד 3 חדרים מתאימים — עם הסבר למה (למשל "החדר הקבוע שלך").
3. אין חדר פנוי? תקבלו:
   - **חלופות** — אותו חדר בשעה קרובה או ביום סמוך.
   - **הצעות החלפה** — "אם דנה תעבור לחדר 4, חדר 1 יתפנה לך" — נשלחת בקשה
     לדנה, והיא מחליטה.
   - **רשימת המתנה** — נרשמים, וברגע שמתפנה חדר מתאים תקבלו התראה.
4. **שעה קבועה** (כל שבוע) — הבקשה עוברת לאישור הניהול; תקבלו התראה כשאושרה.

### היעדרויות
שלושה סוגים:

| סוג | דוגמה | איך |
|---|---|---|
| חד-פעמי / חופשה | "בחופש בין ה-3 ל-10 באוגוסט" | טאב "חד-פעמי", טווח תאריכים |
| שעות מתוך יום | "מחר אני בהשתלמות עד 11:00" | טאב "חד-פעמי" + שעות |
| צמצום קבוע | "כל יום שני אני מסיים ב-14:00" | טאב "צמצום קבוע" |

אפשר גם **לכתוב בשפה חופשית** ("ביום שלישי הקרוב אני יוצאת בשלוש") —
המערכת תבין ותציג לאישור.

החלון שהתפנה מוצע אוטומטית למטפלים אחרים. אם מישהו כבר שובץ בו —
אי אפשר לבטל את ההיעדרות לבד, פונים לניהול.

### התראות
כל מה שרלוונטי אליכם: בקשות החלפה (עם כפתורי אישור/דחייה), אישורי הזמנה,
שינויים שהניהול עשה בלו"ז שלכם, והתראת "התפנה חדר" מרשימת ההמתנה.

כדי לקבל התראות גם כשהאפליקציה סגורה — אשרו את בקשת ההתראות בכניסה
(באייפון: רק אחרי הוספה למסך הבית).

## טיפים

- ההזמנות והבקשות שלכם — תמיד דרך המערכת, כך החדר באמת שמור לכם.
- קיבלתם בקשת החלפה? אשרו או דחו בהקדם — מישהו מחכה לתשובה.
- טעיתם? כמעט לכל פעולה יש כפתור "ביטול" בהודעה שקופצת מיד אחריה.
- משהו לא מסתדר? מנהל/ת המרפאה יכול/ה לערוך הכול.
